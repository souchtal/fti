\hypertarget{md_cmake_autotoc_md10}{}\doxysection{Other configurations}\label{md_cmake_autotoc_md10}
To use a certain compiler, set the environment variables {\ttfamily CC} and {\ttfamily FC} to the name of the compiler executables. ~\newline


For instance if it is desired to use the P\+GI compiler, try\+: ~\newline



\begin{DoxyCode}{0}
\DoxyCodeLine{CC=pgcc FC=pgfortran cmake -\/DCMAKE\_INSTALL\_PREFIX:PATH=install/here/fti ..}
\end{DoxyCode}


To use the built-\/in M\+D5 rather than Open\+S\+SL, please configure using\+: ~\newline



\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/DNO\_OPENSSL=true -\/DCMAKE\_INSTALL\_PREFIX:PATH=/install/here/fti ..}
\end{DoxyCode}


On Cray systems, it might be helping to use the cmake flag\+: ~\newline



\begin{DoxyCode}{0}
\DoxyCodeLine{-\/CMAKE\_SYSTEM\_NAME=CrayLinuxEnvironment}
\end{DoxyCode}
